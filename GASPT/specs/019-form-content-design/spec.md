**기능 번호**: 019
**브랜치**: `019-form-content-design`

**상태**: 명세 완료
**관련 기능**: 017-form-swap-system

---

## 1. 개요

### 1.1 기능 설명
017-폼 교체 시스템의 실제 컨텐츠를 정의합니다. 5종의 플레이어블 폼과 각 폼의 스탯, 스킬, 각성 효과를 구체적으로 설계합니다. GASPT의 원소 마법 테마를 기반으로 다양한 마법 형태를 구현합니다.

### 1.2 설계 원칙
1. **차별화된 플레이 스타일**: 각 폼은 고유한 전투 방식 제공
2. **밸런스**: 어떤 폼도 압도적으로 강하거나 약하지 않음
3. **시너지**: 폼 간 교체 시 보너스 효과로 전략적 플레이 유도
4. **성장 만족**: 각성 시 체감되는 강화
5. **수집 욕구**: 다양한 외형과 스킬로 수집 동기 부여

### 1.3 범위
- **포함**: 폼 5종 상세 데이터, 스킬 10개, 각성 효과, 밸런스 테이블
- **제외**: 폼 구현 코드 (017에서 처리), 아트 에셋 생성

### 2.2 타입별 특성

```
Balance (밸런스)
├─ 평균적인 스탯
├─ 안정적인 스킬
└─ 초보자 친화적

Power (파워)
├─ 높은 공격력
├─ 느린 공격속도
└─ 범위 딜링 특화

Control (컨트롤)
├─ 높은 마나
├─ CC 스킬 보유
└─ 안전한 원거리전

Speed (스피드)
├─ 빠른 공격속도
├─ 높은 이동속도
└─ 연속 콤보 특화

Drain (드레인)
├─ 낮은 체력
├─ 흡혈 효과
└─ 고위험 고보상
```

### 6.1 FormData.cs 확장

```csharp
[CreateAssetMenu(fileName = "FormData", menuName = "GASPT/Forms/FormData")]
public class FormData : ScriptableObject
{
    [Header("기본 정보")]
    public string formId;
    public string formName;
    public string description;
    public FormType formType;       // Balance, Power, Control, Speed, Drain
    public FormRarity baseRarity;   // Common, Rare, Unique, Legendary

    [Header("스탯 (등급별)")]
```
