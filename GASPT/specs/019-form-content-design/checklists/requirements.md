## 5. 테스트 시나리오

### 5.1 기능 테스트
```
시나리오 1: 폼 교체
1. 게임 시작 후 기본 마법사 확인
2. 새 폼 획득
3. Q키로 폼 교체
4. 스탯/스킬 변경 확인
5. 교체 쿨다운 확인

시나리오 2: 각성
1. 동일 폼 2번째 획득
2. 자동 각성 확인
3. 등급 변경 확인
4. 스탯 증가 확인
5. 각성 효과 확인

시나리오 3: 스킬 테스트
1. 각 폼별 스킬 1 발동
2. 데미지 확인
3. 쿨다운 확인
4. 마나/HP 소모 확인
5. 특수 효과 확인 (화상, 빙결 등)
```

### 5.2 밸런스 테스트
```
테스트 1: 스테이지 1 클리어 시간
- 기본 마법사: 목표 5분
- 화염 마법사: 목표 4분 30초
- 얼음 마법사: 목표 5분 30초

테스트 2: 보스 처치 시간
- 각 폼별 스테이지 1 보스 처치 시간 측정
- 편차 30% 이내 확인

테스트 3: 생존률
- 스테이지 1~3 생존률 측정
- 암흑 마법사 > 얼음 마법사 > 기타 순 확인
```
