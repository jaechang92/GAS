## 1. 개요

### 1.1 기능 설명
GASPT의 핵심 시스템 구현이 완료된 후, 시각/청각 피드백 개선, 추가 컨텐츠, 밸런싱 작업을 수행합니다.

### 1.2 범위
- **포함**: 상태 효과 시각화, 사운드 시스템, 추가 폼 컨텐츠, 밸런싱
- **제외**: 새로운 핵심 시스템 추가

---

## 2. 사용자 스토리

### US1: 시각적 피드백 (P1 - 높음)
*"플레이어로서 상태 효과, 데미지, 힐 등이 시각적으로 보이길 원한다"*

**인수 기준**:
- [ ] 화상/빙결/감속 상태 시 캐릭터에 시각 효과 표시
- [ ] 버프/디버프 아이콘에 지속시간 애니메이션 추가
- [ ] 데미지/힐 숫자 이펙트 개선

### US2: 오디오 피드백 (P2 - 중간)
*"플레이어로서 공격, 피격, 스킬 사용 시 효과음을 듣고 싶다"*

**인수 기준**:
- [ ] 공격 시 타격음 재생
- [ ] 피격 시 피격음 재생
- [ ] 스킬 사용 시 스킬별 효과음 재생
- [ ] UI 조작 시 피드백음 재생

### US3: 폼 컨텐츠 확장 (P2 - 중간)
*"플레이어로서 더 다양한 폼으로 변신하고 싶다"*

**인수 기준**:
- [ ] 화염 마법사 폼 추가 (FireStorm, MeteorStrike)
- [ ] 얼음 마법사 폼 추가 (IceLance, FrozenGround)
- [ ] 각 폼별 고유한 플레이 스타일 제공

### US4: 밸런싱 (P3 - 낮음)
*"플레이어로서 어떤 폼을 선택해도 재미있게 플레이하고 싶다"*

**인수 기준**:
- [ ] 폼 간 DPS 차이 ±30% 이내
- [ ] 각 폼의 고유한 강점 유지
- [ ] 치명적인 밸런스 이슈 없음
