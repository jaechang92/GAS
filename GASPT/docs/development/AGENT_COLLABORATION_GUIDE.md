# Agent 협업 가이드

> GASPT 프로젝트의 AI Agent 협업 시스템 가이드

---

## 개요

GASPT 프로젝트는 게임 개발 워크플로우를 최적화하기 위해 전문화된 AI Agent들을 활용합니다. 각 Agent는 특정 영역에 특화되어 있으며, 상호 협업을 통해 효율적인 개발을 지원합니다.

---

## Agent 목록

### 1. gaspt-game-designer
**역할**: 게임 기획 전문가

| 항목 | 내용 |
|------|------|
| **파일** | `.claude/agents/gaspt-game-designer.md` |
| **모델** | opus |
| **주요 기능** | 캐릭터/스킬/적/레벨/아이템 기획서 작성 |
| **협업 대상** | namobanana-2d-art-prompter |

**사용 예시**:
```
"화염 속성 마법사 스킬을 기획해줘"
"1스테이지 보스를 설계해줘"
"대시 스킬을 기획해줘"
```

---

### 2. namobanana-2d-art-prompter
**역할**: 2D 아트 에셋 프롬프트 전문가

| 항목 | 내용 |
|------|------|
| **파일** | `.claude/agents/namobanana-2d-art-prompter.md` |
| **모델** | opus |
| **주요 기능** | namobanana AI용 아트 프롬프트 생성 |
| **출력** | 한글/영문 프롬프트, 애니메이션 가이드, 기술 명세 |

**사용 예시**:
```
"플레이어 캐릭터 스프라이트가 필요해"
"숲 스테이지 배경을 만들어줘"
"체력바 UI 디자인 프롬프트 작성해줘"
```

## 데이터 교환 형식

### 아트 에셋 요청 (기획 → 아트)

```yaml
art_requests:
  - type: character  # character | enemy | background | ui | effect | item
    name: "골렘 전사"
    description: |
      중형 적 캐릭터. 돌로 만들어진 인간형 골렘.
      느리지만 강력한 공격력을 가짐.
    style_guide:
      theme: dark_fantasy
      color_palette: ["#4A4A4A", "#6B6B6B", "#8B4513", "#FF6600"]
    animation_requirements:
      - idle: "서있는 자세, 미세한 호흡"
      - walk: "느린 걸음, 묵직한 발걸음"
      - attack_1: "주먹 내려치기"
      - attack_2: "돌진 공격"
      - hit: "잠깐 움찔"
      - death: "부서지며 쓰러짐"
    resolution: "256x256"
    priority: high
    notes: "관절 분리 가능하도록 디자인 필요"
```

## 사용 시나리오

### 시나리오 1: 새로운 마법 스킬 제작

1. **기획 요청**
   ```
   사용자: "화염 속성 마법 스킬을 만들어줘"
   ```

2. **gaspt-game-designer 응답**
   - 스킬 기획서 (데미지, 쿨다운, 이펙트)
   - 아트 에셋 요청 목록

3. **namobanana-2d-art-prompter 자동 호출**
   - 마법 이펙트 스프라이트 프롬프트
   - 스킬 아이콘 프롬프트

4. **최종 출력**
   - 완성된 기획서
   - 모든 아트 에셋 프롬프트
   - Unity 구현 가이드

---

### 시나리오 2: 새로운 던전 스테이지 제작

1. **기획 요청**
   ```
   사용자: "2스테이지 '얼음 성채' 던전을 기획해줘"
   ```

2. **gaspt-game-designer 응답**
   - 던전 구조 설계
   - 적 배치 계획
   - 보상 테이블

3. **namobanana-2d-art-prompter 자동 호출**
   - 배경 레이어 프롬프트 (원경, 중경, 근경)
   - 타일셋 프롬프트
   - 장식 오브젝트 프롬프트

## Agent 호출 방법

### 직접 호출 (사용자)

특정 Agent를 직접 호출하려면:

```
"[Agent 이름]을 사용해서 [작업 내용]"

예시:
- "gaspt-game-designer를 사용해서 새로운 보스를 기획해줘"
- "namobanana-2d-art-prompter로 UI 버튼 프롬프트 만들어줘"
```

### 자동 호출 (Agent 간)

gaspt-game-designer는 기획서 완료 후 자동으로 namobanana-2d-art-prompter를 호출합니다.

## 문제 해결

### Agent가 호출되지 않을 때
1. Agent 파일이 `.claude/agents/` 폴더에 있는지 확인
2. 파일명과 `name` 필드가 일치하는지 확인
3. YAML frontmatter 형식이 올바른지 확인

### 협업이 원활하지 않을 때
1. 기획서에 `art_requests` 섹션이 포함되어 있는지 확인
2. 요청 형식이 올바른지 확인
