# 1. 플레이어 캐릭터: 암흑 마법사

## 1.1 캐릭터 기획 개요

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 말레피스 (Malefice) |
| **클래스** | 암흑 마법사 (Dark Sorcerer) |
| **테마** | 금지된 마법을 다루는 신비로운 마법사 |
| **성격** | 냉철하고 신비로운, 힘을 추구하는 |
| **외형 컨셉** | 검은 로브, 붉은 눈, 어둠 에너지 |

### 디자인 철학

```
1. 원소 마법사와 대비되는 어두운 비주얼
2. 생명력 소모 기반의 고위험 고보상 플레이
3. 저주와 흡수 계열의 독특한 스킬 메커닉
4. 보라색/붉은색/검은색 컬러 팔레트
```

### 세계관 배경

금지된 마법의 서를 발견하고 암흑의 힘에 눈을 뜬 마법사. 자신의 생명력을 대가로 강력한 마법을 사용한다. 적의 영혼을 흡수하여 자신의 힘으로 삼는 위험하지만 매력적인 마법 스타일.

### 기본 공격: 암흑 화살 (Shadow Arrow)

| 항목 | 내용 |
|------|------|
| **타입** | 원거리 관통 투사체 |
| **속성** | Dark (암흑) |
| **사거리** | 12 유닛 |
| **특수** | 적 2체까지 관통 |

**데미지 공식**:
```
기본 데미지 = ATK * 0.8
관통 시 감소 = 30% (두 번째 적)
흡혈 = 최종 데미지 * VAMP
```

**시각적 연출**:
- 검보라색 화살 형태의 에너지
- 관통 시 잔상 효과
- 적중 시 어둠 입자 흩날림

---

### 스킬1: 생명력 흡수 (Life Drain)

| 항목 | 내용 |
|------|------|
| **타입** | 전방 빔 (채널링) |
| **속성** | Dark (암흑) |
| **범위** | 전방 직선 8 유닛 |
| **지속시간** | 2초 |
| **HP 소모** | 15 |

**데미지 공식**:
```
틱당 데미지 = MAG * 0.8 (0.25초마다)
총 데미지 = MAG * 6.4 (8틱)
HP 회복 = 데미지의 30%
```

**추가 효과**:
- 적에게 준 데미지의 30% HP 회복
- 적 이동속도 20% 감소

**시각적 연출**:
- 캐릭터에서 적으로 이어지는 붉은/검은 에너지 줄기
- 적의 생명력이 빨려나오는 입자 효과
- 캐릭터 주변 붉은 회복 이펙트

---

### 스킬2: 저주의 폭발 (Curse Explosion)

| 항목 | 내용 |
|------|------|
| **타입** | 지정 위치 범위 폭발 |
| **속성** | Dark (암흑) |
| **범위** | 반경 3.5 유닛 원형 |
| **쿨다운** | 10초 |
| **MP 소모** | 40 |

**데미지 공식**:
```
기본 데미지 = MAG * 3.0
저주 상태 적 추가 데미지 = +80%
최대 데미지 = MAG * 5.4
```

**추가 효과**:
- **저주 부여**: 5초간 받는 데미지 20% 증가
- **공포**: 30% 확률로 1초간 도주 (적이 반대 방향으로 이동)

**시각적 연출**:
- 지면에 마법진 생성
- 검보라색 폭발과 함께 해골/유령 형상
- 저주받은 적 머리 위 저주 아이콘

---

### 궁극기: 영혼 수확 (Soul Harvest)

| 항목 | 내용 |
|------|------|
| **타입** | 자신 중심 대규모 범위 |
| **속성** | Dark (암흑) |
| **범위** | 반경 6 유닛 원형 |
| **지속시간** | 3초 |
| **HP 소모** | 50 |
| **MP 소모** | 50 |

**데미지 공식**:
```
틱당 데미지 = MAG * 2.0 (0.5초마다)
총 데미지 = MAG * 12 (6틱)
처치 시 보너스 = HP 20 즉시 회복
```

**추가 효과**:
- 범위 내 적 처치 시 HP 20 회복
- 시전 중 받는 데미지 30% 감소
- 모든 적에게 저주 상태 부여

**시각적 연출**:
- 캐릭터가 공중에 떠오르며 암흑 에너지 방출
- 바닥에 거대한 마법진
- 영혼 형태의 유령들이 적을 공격하는 이펙트
- 처치된 적에서 영혼이 캐릭터로 흡수
