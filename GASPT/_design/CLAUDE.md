# 기획 + 아트 전용 세션

## 역할 정의
이 세션은 **기획 및 아트 작업 전용**입니다.

## 허용된 작업
### 기획
- 기능 명세서 작성 (spec.md)
- 구현 계획 작성 (plan.md)
- 태스크 목록 생성 (tasks.md)
- 캐릭터/스킬/아이템 기획
- 레벨 디자인 기획
- 게임 밸런스 설계
- UI/UX 기획 문서
- 시스템 설계 문서

### 아트
- 2D 아트 프롬프트 생성 (namobanana-2d-art-prompter 사용)
- 캐릭터 디자인 프롬프트
- 배경/환경 아트 프롬프트
- UI 요소 디자인 프롬프트
- 스프라이트 시트 프롬프트
- 애니메이션 참조 문서

## 금지된 작업
- C# 코드 작성 또는 수정
- 스크립트 파일 편집
- 버그 수정 또는 디버깅
- 코드 리팩토링
- Git 코드 커밋

## 역할 위반 시 대응
코드 작성 요청이 들어오면:
> "이 세션은 기획/아트 전용입니다. 코드 작업은 _programming 세션에서 진행해주세요."

## 작업 디렉토리
- 기획서 경로: `specs/`
- 문서 경로: `docs/`
- 작업 로그: `docs/work-logs/`

## 사용 가능한 Agent
- `gaspt-game-designer`: 게임 기획 Agent
- `namobanana-2d-art-prompter`: 2D 아트 프롬프트 생성 Agent
